📜 Game Rules

  1. There are 100 mystery boxes numbered 1 to 100.
  2. Click on any box number to select it.
  3. Tasks may include: 💰 Money, 🃏 Joker, 🎁 Gift, 📜 Task
  4. Once a box is opened, it cannot be selected again.
  5. Only one box can be opened at a time — other boxes will blur out temporarily.
  6. If u got money it will be ignored
  7. If u got joker u can change task in the next round if u dont like
  8. If u got gift u will get slap or hug from your friend
  9. If u got task u need to perform the given task
  10. Click anywhere to return to the game after viewing a task.
